		Online Code Editor

This project is an online code editor built using HTML, CSS, and JavaScript. It allows users to write and execute code in real-time directly in their browser without needing to install any software on their local machine.

		How to Use	

*	Clone or download the repository to your local machine.
*	Open the index.html file in your browser.
*	Start coding! The editor supports HTML, CSS, and JavaScript.
	
		Technologies Used
*	HTML
*	CSS
*	JavaScript
